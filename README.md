# FSM-based Pipeline Validation for RAG Systems

A research project focused on improving the reliability of Retrieval-Augmented Generation (RAG) systems using Finite State Machine (FSM) validation.

## Project Structure
- `src/`: Core logic (FSM, RAG Baseline, RAG+FSM).
- `notebooks/`: Experimentation and results visualization.
- `requirements.txt`: Project dependencies.
- `main_experiment.py`: Script for automated experiments.

## Getting Started
1. Install dependencies: `pip install -r requirements.txt`
2. Run automated experiments: `python main_experiment.py`
3. View results: Open notebooks in the `notebooks/` directory.
